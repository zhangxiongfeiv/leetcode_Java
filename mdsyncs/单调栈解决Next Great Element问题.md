# 单调栈

**栈(stack)** 是一种特殊的线性表，只能在一端进行操作。

- 往栈中添加元素，叫做push，入栈
- 从栈中移除元素，叫做pop，出栈
- 后进先出的原则，Last In First Out (**LIFO**)

## 

**单调栈**实际上就是栈， 只是给**栈做了一些限制**, 使得每次新添加元素后，栈内的元素都保持有序(**单调递增** 或者 **单调递减**)

**单调栈**用途不太广泛，只处理一种典型的问题， 叫做 **Next Greater Element**.
